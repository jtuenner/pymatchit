# Contributing to pymatchit

We welcome contributions! Here is how you can help:

1.  **Report Bugs**: Open an issue on GitHub describing the bug and how to reproduce it.
2.  **Suggest Features**: Open an issue to discuss the feature before implementing it.
3.  **Submit Code**:
    * Fork the repository.
    * Create a new branch (`git checkout -b my-feature`).
    * Install dependencies (`pip install -e .[dev]`).
    * Run tests (`pytest`) to ensure nothing breaks.
    * Open a Pull Request.
